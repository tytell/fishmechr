# Zebrafish body shape

Width and height profile of a zebrafish body as a function of fractional
arc length. Used for estimating volumes and centers of mass.

## Usage

``` r
zebrafish_shape
```

## Format

A data frame with 20 rows and 3 columns:

- s:

  Arc length as a fraction of body length (0 = head, 1 = tail)

- width:

  Lateral body width as a fraction of body length

- height:

  Dorso-ventral body height as a fraction of body length
