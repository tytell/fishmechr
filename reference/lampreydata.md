# Lamprey midline data

Midline tracking data for a sea lamprey (*Petromyzon marinus*) swimming
steadily. Contains 20 evenly-spaced points along the body from head to
tail, across 80 frames.

## Usage

``` r
lampreydata
```

## Format

A data frame with 1600 rows and 5 columns:

- t:

  Time in seconds

- frame:

  Frame number

- point:

  Point index along the body (1 = head, 20 = tail)

- mxmm:

  x coordinate in mm

- mymm:

  y coordinate in mm
